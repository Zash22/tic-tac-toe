# tic-tac-toe
Tic-Tac-Toe app built with Vite using 'vanilla' template

Setup:
1. git clone git@github.com:Zash22/tic-tac-toe.git
2. git checkout dev
3. npm install -g pnpm && pnpm install

To run Playwright tests:
1. pnpm exec playwright test

To run Vitest:
1. npx vitest
